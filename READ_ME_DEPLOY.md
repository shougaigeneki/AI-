# 🚀 WEBサイト公開ガイド

このプロジェクトを24時間いつでもどこからでも使える「本物のWEBサイト」にするための手順です。

## 💡 もっとも簡単な方法：ドラッグ＆ドロップで公開（数分で完了）
GitHubの使い方が難しそうに感じる場合は、この方法が一番おすすめです。

### 手順
1. **`dist` フォルダを準備する**
   - フォルダの中にある `dist` という名前のフォルダを確認してください。これが「本番用」の中身です。
2. **Netlify Dropにアクセス**
   - [Netlify Drop](https://app.netlify.com/drop) をブラウザで開きます。
3. **フォルダを放り込む（ドラッグ＆ドロップ）**
   - 画面の「Drag and drop your site folder here」と書かれた場所に、PCの `dist` フォルダをそのままマウスで引きずって放り込みます。
4. **公開完了！**
   - 数秒で `https://...netlify.app` というURLが表示されます。これがあなたのサイトの住所です！

## 🚀 GitHubを使ってVercelで公開（アカウントをお持ちの方用）
GitHubに不慣れな場合でも、ブラウザだけで完結する一番簡単な方法をご案内します。

### 手順
1. **GitHubで新しい場所（リポジトリ）を作る**
   - [GitHub](https://github.com/new) にアクセスし、名前に「my-search-site」などと入力して「Create repository」を押します。
2. **ファイルをブラウザでアップロードする**
   - 画面の真ん中あたりにある「**uploading an existing file**」という青いリンクをクリックします。
   - このGoogle Driveフォルダの中身（`src`, `public`, `index.html`, `package.json` など。※`node_modules`は不要です）を、ブラウザの画面に**全部まとめてドラッグ＆ドロップ**します。
   - 画面下の「Commit changes」ボタンを押して完了です。
3. **Vercelに読み込ませる**
   - [Vercel](https://vercel.com/new) にアクセスし、GitHubアカウントでログイン。
   - 先ほど作った「my-search-site」を選んで「Import」→「Deploy」を押すだけです！
4. **公開完了！**
   - 数分で全世界に公開されます。スマホからも見れるようになります。

## 🔒 セキュリティとコストの仕組み（重要）

このサイトは**「利用者各自が自分のAPIキーを持ち寄る（BYOK）」**という安全な設計になっています。

- **主催者のコストはゼロ**: あなたがサイトを公開しても、他人が使った分のAI使用料があなたに請求されることはありません。利用者は自分のキーを入力して使う必要があるからです。
- **データはブラウザ内のみ**: 入力されたキーは、その人のスマホやPCのブラウザ内にのみ保存されます。ネット上に流出することはありません。
- **スマホでも安心**: あなた自身がスマホから使うときも、一度自分のキーを入力すれば、それ以降はあなたの無料枠の範囲内で安全に利用できます。

## 🤖 なぜGemini（AI）を使うのか？

普通の検索サイト（Googleなど）との違いは、**「AIによる分析と評価」**にあります。

1. **自動検品**: 検索結果から「単なる広告」や「質の低い情報」をAIが自動で排除します。
2. **情報の要約**: 複数のサイトを見比べなくても、AIが要点をまとめて「一生モノ」の候補を提案してくれます。
3. **トレンド把握**: 今何が本当に評価されているのかを、膨大な検索データの中からAIが読み解きます。

単に検索するだけでなく、**「AIが代わりに調べて、答えを教えてくれる」**のがこのツールの最大のメリットです。
